# SmartBudget Tracker – Claude AI Version

## Overview
SmartBudget Tracker is an AI-powered personal finance management web application developed as part of the IT4020 – Modern Topics in IT assignment. This version was generated and refined using **Claude AI (Anthropic)**.

## Features
- **Transaction Management** – Add, view, and delete income/expense transactions with category tagging
- **Summary Dashboard** – Real-time display of total income, total expenses, current balance, and savings rate
- **Category Breakdown Chart** – Dynamic donut/pie chart rendered using the HTML5 Canvas API
- **Spending Alerts** – Automatic warnings when any expense category exceeds 80% of its budget limit
- **AI Financial Advisor** – Data-driven financial feedback and improvement suggestions
- **Data Persistence** – All transactions saved to browser localStorage

## How to Run
1. Open `index.html` in any modern web browser
2. No build tools or server required – runs entirely in the browser
3. Transactions are auto-saved to localStorage
